# Just A Forum

## Description

### Minimalist, Responsive forum website

## Deployement

Frontend: React.js, ~~MUI~~ Tailwind

- Installed locally: run: `npm run dev` to start
- Hosted by Netlify

Backend: Ruby on Rails  
Database: PostgreSQL

- Hosted by Heroku
